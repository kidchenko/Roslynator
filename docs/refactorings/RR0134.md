## Invert prefix/postfix unary operator

| Property           | Value                                |
| ------------------ | ------------------------------------ |
| Id                 | RR0134                               |
| Title              | Invert prefix/postfix unary operator |
| Syntax             | prefix/postfix unary expression      |
| Span               | operator token                       |
| Enabled by Default | &#x2713;                             |

### Usage

#### Before

```csharp
int i = 0;

i++;
```

#### After

```csharp
int i = 0;

i--;
```

- - -

#### Before

```csharp
int i = 0;

++i;
```

#### After

```csharp
int i = 0;

--i;
```
